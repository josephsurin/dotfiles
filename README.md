# dotfiles

Dotfiles for my Arch Linux system running on my Thinkpad X230

### Programs:
  - Window Manager: i3-gaps
  - Status Bar: polybar
  - Window Switcher / Application Menu: rofi
  - Terminal: Termite
  - File Manager: ranger
  - PDF viewer: zathura
  - Browser: Firefox
  - Text Editors: vim / vscode
  - Image Viewer: feh / sxiv
  - Screenshots: maim + xclip
